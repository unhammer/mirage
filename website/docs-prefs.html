Documentation

Preferences

Open Behavior

Open new image in - Defines the default view that a new image is loaded with. Smart Mode uses 1:1 zooming if the image is smaller than the available space and Fit Mode if the image is larger. Last Active Mode will use whichever zooming action the user last manually chose from the menu or toolbar. [Default: Smart Mode]

Load all images in current directory - If selected, when an image is loaded in Mirage, all other images in that directory will also be loaded. This mimics the behavior of some other image viewers. [Default: Unchecked]

Use ... directory - Determines what default directory should be used for the File > Open dialog. Either the last navigated directory or a fixed directory (i.e. the base directory to a user's images). [Default: Use last chosen directory]

Navigation

Wrap around imagelist - If set to Yes, choosing Next Image from the last image in the list will wrap around to the first image in the list (and vice versa from the first image in the list. If set to Prompt User, the user will be able to choose from a dialog whether they wrap or not. [Default: No]

Preload images for faster navigation - If enabled, Mirage will load the previous and next images into memory in order to speed up navigation. Most users will find that the responsiveness of the application dramatically improves with navigating large or high- quality images. However, because this feature increases memory usage, it is not recommended on machines with limited amount of ram. [Default: Checked]

Interface

Background color - Allows the user to specify the background color of the application. [Default: Black]

Open Mirage in fullscreen mode - Automatically starts Mirage in fullscreen mode. The user can leave fullscreen mode by pressing Escape, Shift-Enter, or right-clicking on the image. [Default: Unchecked]

Slideshow Mode

Delay between images - Specifies the delay between automatically loading images in slideshow mode. This value can be overridden on a one-time basis in fullscreen mode via the controls that appear when the mouse approaches the bottom of the screen. Note that the delay specifies the time between the display of the previous image and the beginning of the next image's loading; therefore the delay may be slightly longer. [Default: 3 seconds]

Disable screensaver in slideshow mode - Temporarily disables xscreensaver during slideshows. [Default: Checked]

Always start in fullscreen mode - If enabled, the application will automatically be put into fullscreen mode when slideshow mode is started. [Default: Unchecked]

Randomize order of images - Instead of automatically proceeding to the next image in the list, a random image will be displayed. Note that, similar to selecting Go > Random Image, Mirage will attempt to only display each image once before images are repeated. [Default: Unchecked]

Image Editing

Scaling quality - Allows the user to specify what scaling algorithm is used for resizing and zooming images. Actions that produce better quality will also take longer, and vice versa. [Default: Bilinear]

Modified images - Specifies whether any modifications to images should be automatically saved, ignored, or prompt the user for a decision. Note that only certain image formats are supported for saving (jpg, png, tiff, and bmp). [Default: Prompt for Action]

Confirm image delete - If checked, prompts the user to confirm that they really want to delete an image. Note that deleting files is a permanent action and cannot be undone.
