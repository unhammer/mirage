Requirements:
   (Required) PyGTK 2.6.0 or newer
   (Required) GTK 2.8.0 or newer
   (Optional) Gnome-python if you want Mirage's toolbar to follow your Gnome setting
   (Building) GCC
   (Building) python-dev (on some distros)

Mirage source code is available here:  mirage-0.7.3.tar.gz   mirage-0.7.3.tar.bz2

   Ubuntu: package here or here (Dennis Kaarsemaker)
   Archlinux: Enable community and pacman -S mirage (William Rea)
   Gentoo: You can find an ebuild here (David Shakaryan)
   Frugalware: Enable extra and pacman -S mirage (Priyank)
   Foresight Linux: package here (Paul Scott-Wilson)
   Crux: ports DB (Jared Housh)

Mirage uses subversion for its development version control. You can checkout the latest code with:
          svn checkout http://svn.berlios.de/svnroot/repos/mirageiv/trunk mirage
You can also browse the latest code with WebSVN
