Mirage uses subversion for its development version control. You can checkout the latest code with:
          svn co http://svn.berlios.de/svnroot/repos/mirageiv/trunk mirage
You can also browse the latest code with WebSVN

Requirements

PyGTK 2.6.0 or newer
GTK 2.6.0 or newer
Gnome-python to have the toolbar to follow your Gnome setting (optional)
GCC (building only)
python-dev (on some distros; building only)
